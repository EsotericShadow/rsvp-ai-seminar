Database Schema Overview

Core Models

Campaign
• id: String (Primary Key)
• name: String
• description: String?
• status: CampaignStatus
• createdAt: DateTime
• updatedAt: DateTime
RSVP
• id: String (Primary Key)
• firstName: String
• lastName: String
• email: String
• phone: String?
• attendanceStatus: String
• attendeeCount: Int

Relationship Models

AudienceGroup
• id: String (Primary Key)
• name: String
• description: String?
• color: String?
• members: AudienceMember[]
CampaignSchedule
• id: String (Primary Key)
• campaignId: String (FK)
• templateId: String (FK)
• groupId: String (FK)
• scheduledFor: DateTime?
• status: CampaignSendStatus
